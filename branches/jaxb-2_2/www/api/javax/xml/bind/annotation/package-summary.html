Package javax.xml.bind.annotation Description

Defines annotations for customizing Java program elements to XML Schema mapping.

Package Specification

Terminology

JavaBean property and public, non final non static field: For the purposes of mapping, there is no semantic difference between a public non final, non static field and a JavaBean property. Thus, an annotation that can be applied to a JavaBean property can always be applied to a public non static, non final field. Hence in the Javadoc documentation, for brevity, the term JavaBean property is used to mean either JavaBean property or a public non final, non static field. Where required, both are explicitly mentioned.

mapping annotation:A JAXB 2.0 defined program annotation based on the JSR 175 programming annotation facility.
